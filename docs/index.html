A Minimalist API CMS

Nuri is a lightweight API Content Management System that stores data in files, eliminating the need for a database. It's simple, flexible, and perfect for modern web applications.

View on GitHub

Features

  • Lightweight & No Database: Nuri uses SQLite to store data in files, removing the complexity of managing a database.
  • API for Content Delivery: Nuri provides an API to deliver content efficiently to applications, websites, and blogs.
  • Flexible Content Modeling: Define your own content structure and adapt it to any use case.

Get Started

  1. Clone the repository: git clone https://github.com/janmarkuslanger/nuri
  2. Install dependencies: pip install -r requirements.txt
  3. Run the application: python run.py
