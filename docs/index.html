Features

Lightweight & No Database

Nuri uses SQLite to store data in files, removing the complexity of database management.

API for Content Delivery

Effortlessly deliver content through an API tailored for applications, websites, and blogs.

Flexible Content Modeling

Easily define and adapt your content structure to fit any use case.

Get Started

  1. Clone the repository: git clone https://github.com/janmarkuslanger/nuri
  2. Install dependencies: pip install -r requirements.txt
  3. Run the application: python run.py
