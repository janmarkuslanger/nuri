A Minimalist API CMS

Nuri is a lightweight API Content Management System that stores data in files instead of databases, making it easy to set up and maintain.

View on GitHub

Features

File-Based Storage

All content is stored in JSON files, eliminating the need for a database.

RESTful API

Access and manage content through a simple and flexible API.

Lightweight Design

Minimal dependencies and straightforward configuration.

Get Started

  1. Clone the repository: git clone https://github.com/janmarkuslanger/nuri
  2. Install dependencies: pip install -r requirements.txt
  3. Run the application: python run.py
